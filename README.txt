====================================================
  YouTube to MP3 Downloader - Οδηγίες
====================================================

ΠΕΡΙΕΧΟΜΕΝΑ ΦΑΚΕΛΟΥ
--------------------
  yt_downloader.py     → Ο κώδικας της εφαρμογής
  yt_downloader.spec   → Ρυθμίσεις για το build
  requirements.txt     → Βιβλιοθήκες Python
  BUILD.bat            → Script για δημιουργία .exe
  README.txt           → Αυτό το αρχείο


ΒΗΜΑ 1 - Εγκατάσταση Python
-----------------------------
Αν δεν έχεις Python:
  → https://www.python.org/downloads/
  ΣΗΜΑΝΤΙΚΟ: Στην εγκατάσταση τσεκάρε το
  "Add Python to PATH"


ΒΗΜΑ 2 - Κατέβασμα ffmpeg
---------------------------
  1. Πήγαινε στο: https://www.gyan.dev/ffmpeg/builds/
  2. Κατέβασε το: ffmpeg-release-essentials.zip
  3. Άνοιξε το zip → φάκελος bin
  4. Αντίγραψε τα:
       • ffmpeg.exe
       • ffprobe.exe
     μέσα σε αυτόν τον φάκελο (δίπλα στο BUILD.bat)


ΒΗΜΑ 3 - Δημιουργία .exe
--------------------------
  Κάνε διπλό κλικ στο BUILD.bat
  Περίμενε 1-2 λεπτά μέχρι να τελειώσει.

  Το αποτέλεσμα θα βρίσκεται στον φάκελο OUTPUT:
    OUTPUT\
      YT_to_MP3.exe
      ffmpeg.exe
      ffprobe.exe


ΒΗΜΑ 4 - Χρήση
---------------
  Αντίγραψε ολόκληρο τον φάκελο OUTPUT όπου θέλεις.
  Τρέξε το YT_to_MP3.exe

  ΣΗΜΑΝΤΙΚΟ: Τα ffmpeg.exe + ffprobe.exe πρέπει
  να είναι ΠΑΝΤΑ στον ίδιο φάκελο με το .exe!


ΣΥΧΝΑ ΠΡΟΒΛΗΜΑΤΑ
-----------------
  • "ffmpeg δεν βρέθηκε"
    → Βεβαιώσου ότι ffmpeg.exe + ffprobe.exe
      είναι δίπλα στο YT_to_MP3.exe

  • "Αποτυχία λήψης"
    → Έλεγξε αν το URL είναι σωστό
    → Ορισμένα βίντεο δεν επιτρέπουν λήψη

  • Το BUILD.bat δεν ανοίγει σωστά
    → Κάνε δεξί κλικ → "Εκτέλεση ως διαχειριστής"
